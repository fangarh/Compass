# AI

Эта папка - рабочая память Codex по проекту.

Здесь можно хранить:

- краткую карту кодовой базы;
- текущее состояние проекта;
- журнал изменений и синхронизации;
- технические выводы после чтения кода;
- открытые риски и следующие шаги.

Папка не обязана быть публичной документацией. Если вывод важен для людей, его нужно вынести в соседние разделы Obsidian: `Архитектура`, `Решения`, `Проверки`, `Материалы`.
