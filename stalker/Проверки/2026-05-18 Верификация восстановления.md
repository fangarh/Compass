# 2026-05-18 Верификация восстановления

## Что проверено

### APK decode

- Apktool 3.0.2 успешно декодировал `compassv33.apk`.
- JADX 1.5.5 экспортировал Java-like source с несколькими decompiler errors.
- Исходники пакета `net.afterday.compas` перенесены в `app/src/main/java/net/afterday/compas`.

### Java recovery build

Прямая сборка JADX-исходников сначала падала из-за типичных проблем декомпиляции:

- synthetic lambda-классы;
- enum switch helper arrays;
- duplicate/unresolved fields and locals;
- старые support imports;
- зависимость `java8.util.Optional`.

Вывод: JADX полезен как материал восстановления, но требовал ручной чистки.

### Modern Gradle debug build

Debug APK был успешно собран.

## Известное предупреждение

Android throttles once-per-second Wi-Fi scans.
Это не crash, но radio detection нужно проектировать с учетом системных лимитов.

## Baseline rebuild

Apktool baseline rebuild также прошел и дал rebuild APK на базе original manifest values.
Этот APK нужен как бинарный recovery baseline, а не как целевая modern-сборка.
