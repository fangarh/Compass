# Пользовательское руководство

Основной материал сейчас находится в репозитории:

- `docs/user-guide/pda-compass-user-guide.md`;
- изображения: `docs/user-guide/images/`.

Документ описывает:

- назначение PDA Compass;
- первый запуск и разрешения Android;
- главный экран;
- QR-сканер;
- инвентарь и карточку предмета;
- настройки;
- foreground service notification;
- состояния персонажа;
- уровни PDA;
- игровые влияния и Wi-Fi/Bluetooth/GPS источники;
- режимы Wi-Fi-сканирования;
- советы для игроков и организаторов;
- ограничения восстановленной версии.

Важное техническое замечание: файл корректно читается как UTF-8. В Windows PowerShell без явного `-Encoding UTF8` русские Markdown-файлы могут отображаться как mojibake, поэтому при проверке содержимого нужно задавать UTF-8 явно.
